 # ICED PLAYGROUND
personal playground for learning and building UIs with iced.rs


---

create a new iced application

```sh
cargo new your_iced_application_name
cd your_iced_application_name
cargo add iced
```

other helpful cargo commands
```sh
cargo run
cargo check
cargo clean
```
